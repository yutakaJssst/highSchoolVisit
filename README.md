# AIと作る高校紹介サイト

高校説明会で、AIを使ったWebサイト制作をその場で見せるためのGitHub Pages向けテンプレートです。
ビルド手順は不要で、`index.html`をブラウザで開くだけで動きます。

## ファイル構成

- `index.html`: ページ構造
- `styles.css`: 見た目とレスポンシブ対応
- `script.js`: 学校データ、描画、ミニゲーム

## 当日の使い方

1. 生徒に学校の好きなところ、色、行事、部活動などを聞きます。
2. `script.js`冒頭の`siteData`を書き換えます。
3. ブラウザを再読み込みして、サイトが変わる様子を見せます。
4. 余裕があれば、`features`、`voices`、`events`、`game`を追加変更します。

## GitHub Pagesで公開

1. GitHubでこのリポジトリを開きます。
2. `Settings`から`Pages`を開きます。
3. `Build and deployment`で`Deploy from a branch`を選びます。
4. Branchを`main`、Folderを`/ (root)`にして保存します。

## 実演用プロンプト例

```text
この高校の名前を「〇〇高等学校」にして、学校の魅力を
「部活動が盛ん」「駅から近い」「探究学習が面白い」に変えてください。
テーマカラーは緑とオレンジで、文化祭を目立たせてください。
```

```text
ミニゲームを、学校のマスコットがアイデアを集めるゲームに変えてください。
障害物は締切、アイテムはひらめきにしてください。
```
